# ベタ基礎ツール

事前に描いた2D図形を基準に、ベタ基礎の3D図形を（基礎パッキン、断熱材、捨てコン、砕石を含む）を一括作成するツールです。

使い方はサンプルを参考に。


・各種構成を変えたい場合はコードを書き換えてください。
・各種寸法・長さ・面積を2D図形から取得すれば、根切りからコンクリートや型枠、鉄筋などの数量の概算を計算させることもできるかと思います。
